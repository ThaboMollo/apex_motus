# Apex Motus Rebuild Plans

This folder contains the implementation plans to rebuild the website from the `ApexMotus_spec.md` foundation.

Plan files:

1. `01-master-implementation-plan.md`
2. `02-teardown-and-theme-retention-plan.md`
3. `03-execution-checklist-and-milestones.md`

Scope directive reflected in these plans:

- Rebuild the site from scratch.
- Keep only theming foundations (colors, typography, visual tokens, logos) from the current codebase.
- Replace current pages/components with the new landing page structure in the spec.

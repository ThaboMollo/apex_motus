Compact, stays small

Clean and familiar. But users typing 5 facts (often 2-4 sentences) quickly feel cramped — text scrolls out of view and they can't see what they've written.

Works well if each fact is short (one phrase). Struggles with longer input.

Expands as you type

Auto-grows from 1 line to ~6 lines. Users can see all their facts at once, edit freely, and add more. Send button is inside the box at the bottom right — like claude.ai.

Better for quality input. Users who can see what they've written tend to add more — richer input = better diagnostic.
